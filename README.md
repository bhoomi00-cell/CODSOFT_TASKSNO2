# CodSoft Python Internship - Task 2: Calculator

## Description
A simple command-line based calculator application built using Python. 
The program takes two numbers and an arithmetic operator as input from 
the user and performs the calculation accordingly.

## Features
- Supports basic arithmetic operations: Addition, Subtraction, 
  Multiplication, Division
- Handles invalid operator input
- Handles division by zero error
- Accepts decimal (float) numbers

## Technologies Used
- Python 3

## How to Run
1. Clone this repository
2. Run the file using: `python calculator.py`
3. Enter two numbers and choose an operation when prompted

## Sample Output
Enter a number: 10
Enter other number: 5
Enter operation to perform: +
15.0
